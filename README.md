# SmartFit Pro - Product Landing Page
A fully responsive product landing page built with semantic HTML5 and modern CSS3.
This project was developed to demonstrate front-end fundamentals inculding layout structuring, responsive design, fixed navigation systems, form validation, and media queries. The implementation satisfies all freeCodeCamp Product Landing Page user stories while maintaining clean code structure and custom styling.

## Project Objective
To build a fully functional, responsive product landing page that:
- Uses semantic HTML structure
- Implements a fixed navigation bar
- Enables smooth anchor-based navigation
- Embeds a product video
- Inculdes an email subscription form with HTML5 validation
- Uses CSS Flexbox for layout
- Applies responsive design principles via media queries

## Technical Implementation Highlights

### Layout Strategy
- Flexbox used for navigation alignment and content sections
- `position: fixed` used for persistent header
- Margin offset applied to prevent content overlap with fixed header

### Responsiveness
- Media query applied for screens below 768px
- Navigation layout shifts from horizontal to vertical stacking
- Flexible wrapping enabled using `flex-wrap`

### Form Validation
- Native HTML5 validation using:
  - `type="email"`
  - `required` attribute
- Form submission directed to static endpoint as required
